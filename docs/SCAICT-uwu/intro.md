# 快速入門

## 這是什麼？

中電喵是一隻住在中電會 Discord 社群的一隻貓咪，喜歡唱、跳、coding、算數。你可以透過斜線指令 (slash command) 或在專用頻道和它互動

### 怎麼使用

- 加入伺服器 link
- 領取身分組
- 到 #每日充電 使用 /charge 拿到屬於你的第一份電電點
- 在伺服器享受所有中電喵的服務
