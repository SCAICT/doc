## 你有有多少電電點
> ```/check_point```
- 限制使用權限：false


無須攜帶任何參數，會檢索指令發起人當前的連續登入、電電點數量狀態

## 每日充電
> ```/charge```

- 限制使用權限：false

```/charge```是獲得電電點最簡單的管道，每天可以到 ```#每日充電``` 頻道使用這個命令，會直接拿到 5 電電點，連續登入的話或許會有特別的獎勵？
